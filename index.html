Python dépasse enfin C et Java

Pour la première fois depuis plus de 20 ans, Python prend la tête du classement des langages de programmation les plus populaires avec 11,27 % devant C (11,16 %) et Java (10,46 %). Paul Jansen, le PDG de TIOBE Software explique que « sa facilité d’apprentissage, son énorme quantité de bibliothèques et son utilisation répandue dans toutes sortes de domaines en ont fait le langage de programmation le plus populaire d’aujourd’hui ». Peu surprenant au vu des tendances précédentes, ce changement marque la perte de notoriété des langages vieillissants comme C ou Java. À l’inverse, Python (déjà leader dans des secteurs qui explosent comme le data mining ou l’intelligence artificielle) gagne en popularité de manière relativement constante.
